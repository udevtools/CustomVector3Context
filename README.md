Create a json file named "Vector3Actions.json" in the Unity Resources folder.
Add the following JSON string inside this file:

{
  "actions": [
    {
      "name": "Zero",
      "values": [ 0, 0, 0 ]
    },
    {
      "name": "One",
      "values": [ 1, 1, 1 ]
    },
    {
      "name": "Ten",
      "values": [ 10, 10, 10 ]
    }
  ]
}

You can Add, Remove or Modify actions to customize Vector3 context menu.
